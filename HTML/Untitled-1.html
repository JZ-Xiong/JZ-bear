HTML5 + CSS3 初学者实验室

HTML5 核心标签

语义化标签

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>语义化文档结构</title>
</head>
<body>
    <header>
        <h1>网站标题</h1>
        <nav>
            <ul>
                <li><a href="#">首页</a></li>
                <li><a href="#">关于</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>文章标题</h2>
            <p>文章内容...</p>
        </article>
        
        <aside>
            <h3>相关链接</h3>
            <ul>
                <li><a href="#">参考1</a></li>
            </ul>
        </aside>
    </main>
    
    <footer>
        <p>© 2023 我的网站</p>
    </footer>
</body>
</html>
学习重点： 语义化标签(header, nav, main, article, aside, footer)提升SEO和可访问性，使代码结构更清晰

表单增强

<form>
    <label for="email">电子邮箱:</label>
    <input type="email" id="email" required placeholder="输入有效邮箱">
    
    <label for="date">选择日期:</label>
    <input type="date" id="date">
    
    <label for="range">音量控制:</label>
    <input type="range" id="range" min="0" max="100">
    
    <label for="color">选择颜色:</label>
    <input type="color" id="color">
    
    <button type="submit">提交</button>
</form>
学习重点： HTML5新增的表单类型(email, date, range, color等)提供更好的输入控制和验证

响应式布局技术

Flexbox 布局

.container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
    align-items: center;
    gap: 1rem;
}

.item {
    flex: 1 1 200px;
    background: #4361ee;
    padding: 1rem;
    border-radius: 8px;
    color: white;
}
学习重点： Flexbox 提供一维布局解决方案，适合组件和小规模布局

Grid 布局

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1.5rem;
}

.grid-item {
    background: #3f37c9;
    padding: 1.5rem;
    border-radius: 8px;
    color: white;
}
学习重点： CSS Grid 是二维布局系统，适合整个页面的复杂布局

媒体查询

/* 默认样式 */
.responsive-box {
    width: 100%;
    padding: 1rem;
    background: #4cc9f0;
}

/* 平板设备 */
@media (min-width: 768px) {
    .responsive-box {
        width: 80%;
        margin: 0 auto;
    }
}

/* 桌面设备 */
@media (min-width: 1024px) {
    .responsive-box {
        width: 60%;
    }
}
学习重点： 使用媒体查询根据屏幕尺寸应用不同样式，实现响应式设计
