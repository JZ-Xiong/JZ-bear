HTML 基础结构

标准HTML文档骨架定义了网页的基本结构，每个元素都有特定的语义作用。

HTML基本结构 
<!DOCTYPE html>                     <!-- 文档类型声明，确保浏览器使用标准模式 -->
<html lang="zh-CN">              <!-- 根元素，定义页面语言为中文 -->
<head>
    <meta charset="UTF-8">        <!-- 字符编码声明，支持中文 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 移动端视口设置 -->
    <title>页面标题</title>        <!-- 浏览器标签标题，影响SEO和收藏夹 -->
    <link rel="stylesheet" href="styles.css"> <!-- 外部CSS链接 -->
</head>
<body>
    <!-- 页面内容区域 -->
    <script src="script.js"></script> <!-- 外部JS脚本，通常放在body末尾 -->
</body>
</html>
语义化标签示例

<nav>导航菜单</nav>           <!-- 导航区域，包含网站的主要链接 -->
<main>                       <!-- 主要内容容器，每个页面应该只有一个 -->
    <article>                <!-- 独立文章内容，可独立分发 -->
        <h1>文章标题</h1>
        <section>            <!-- 内容区块，通常有标题 -->
            <h2>章节标题</h2>
            <p>段落内容...</p>
        </section>
    </article>
    <aside>侧边栏</aside>     <!-- 相关内容/侧边栏，与主要内容间接相关 -->
</main>
<footer>网站底部</footer>     <!-- 页脚内容，通常包含版权和联系信息 -->

CSS 选择器

选择器用于定位 DOM 元素并为其应用样式，掌握选择器语法和优先级对写出健壮的样式非常重要。

常见选择器类型 
/* 元素选择器 */
span { color: red; }

/* 类选择器 */
.btn { padding: 8px 12px; }

/* ID 选择器（优先级高） */
#main-title { font-size: 2rem; }

/* 后代/子选择器 */
.nav a { color: #333; }    /* 后代 */
.nav > a { color: #222; }  /* 直接子元素 */

/* 相邻兄弟 / 通用兄弟 */
h1 + p { margin-top: 0.5rem; }    /* 紧邻 */
h1 ~ p { color: #666; }          /* 之后所有兄弟 */

/* 属性选择器 */
input[type="text"] { width: 100%; }

/* 伪类与伪元素 */
a:hover { text-decoration: underline; }
input::placeholder { color: #999; }
选择器优先级示例

常见规则：ID > 类/伪类 > 元素。
例如：#id .class element 的权重高于单独的 .class。

盒模型 (Box Model)

理解内容(content)、内边距(padding)、边框(border)、外边距(margin) 的计算方式，以及 box-sizing 如何影响布局。

盒模型示例 
/* 默认 box-sizing: content-box (宽度/高度只包含 content) */
.box { width: 200px; padding: 20px; border: 4px solid #ccc; margin: 16px; }

/* 推荐使用 border-box，宽度包含 border 和 padding，更直观 */
*, *::before, *::after { box-sizing: border-box; }
盒模型视觉化
内容区域 (content)

调整 padding/border 会影响元素的占位，使用 box-sizing: border-box 常能避免溢出。

CSS 过滤器 (Filters)

过滤器允许对元素或图片进行图形效果处理，如模糊、亮度、对比度等，适用于图像处理和视觉增强。

常用 filter 
img { filter: grayscale(100%); }
img:hover { filter: none; }

响应式设计

响应式设计确保网站在各种设备上都能提供最佳用户体验。

媒体查询基本语法
/* 移动设备优先 (默认样式) */
.container { 
    padding: 10px; 
}

/* 平板设备 (最小宽度768px) */
@media (min-width: 768px) {
    .container { 
        padding: 20px; 
    }
}

/* 桌面设备 (最小宽度1024px) */
@media (min-width: 1024px) {
    .container {
        padding: 30px;
        max-width: 1200px;
        margin: 0 auto; /* 居中显示 */
    }
}
响应式图片

CSS 变量（自定义属性）

CSS变量提供了一种在整个文档中重用和动态修改CSS值的方法。

CSS变量定义和使用 
/* 在根元素定义全局变量 */
:root {
    --primary-color: #4361ee;
    --spacing-unit: 8px;
    --max-width: 1200px;
}

/* 使用CSS变量 */
.button {
    background-color: var(--primary-color);
    padding: var(--spacing-unit);
}

/* 局部作用域变量 */
.card {
    --card-color: blue;
    background: var(--card-color);
}

/* 使用回退值 */
.element {
    color: var(--undefined-color, black);
}

CSS 变换与过渡

CSS提供了强大的变换和过渡效果，可以创建流畅的动画和交互效果。

2D/3D 变换 
/* 2D 变换 */
.element {
    transform: translate(50px, 100px);    /* 位移 */
    transform: rotate(45deg);             /* 旋转 */
    transform: scale(1.5);                /* 缩放 */
    transform: skew(10deg, 20deg);        /* 倾斜 */
}

/* 3D 变换 */
.element-3d {
    transform: rotateX(45deg);            /* X轴旋转 */
    transform: rotateY(45deg);            /* Y轴旋转 */
    transform: translateZ(100px);         /* Z轴位移 */
    transform-style: preserve-3d;         /* 3D空间 */
    perspective: 1000px;                  /* 透视距离 */
}
