HTML 基础结构

标准HTML文档骨架定义了网页的基本结构，每个元素都有特定的语义作用。

HTML基本结构 
<!DOCTYPE html>                     <!-- 文档类型声明，确保浏览器使用标准模式 -->
<html lang="zh-CN">              <!-- 根元素，定义页面语言为中文 -->
<head>
    <meta charset="UTF-8">        <!-- 字符编码声明，支持中文 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 移动端视口设置 -->
    <title>页面标题</title>        <!-- 浏览器标签标题，影响SEO和收藏夹 -->
    <link rel="stylesheet" href="styles.css"> <!-- 外部CSS链接 -->
</head>
<body>
    <!-- 页面内容区域 -->
    <script src="script.js"></script> <!-- 外部JS脚本，通常放在body末尾 -->
</body>
</html>
语义化标签示例
<header>网站头部</header>       <!-- 页面头部，通常包含logo和主要导航 -->
<nav>导航菜单</nav>           <!-- 导航区域，包含网站的主要链接 -->
<main>                       <!-- 主要内容容器，每个页面应该只有一个 -->
    <article>                <!-- 独立文章内容，可独立分发 -->
        <h1>文章标题</h1>
        <section>            <!-- 内容区块，通常有标题 -->
            <h2>章节标题</h2>
            <p>段落内容...</p>
        </section>
    </article>
    <aside>侧边栏</aside>     <!-- 相关内容/侧边栏，与主要内容间接相关 -->
</main>
<footer>网站底部</footer>     <!-- 页脚内容，通常包含版权和联系信息 -->

HTML 文本元素

HTML提供了多种文本元素，用于创建结构化的内容层次。

文本格式化标签

<hr>                <!-- 水平分隔线，主题分隔 -->
<blockquote>引用文本</blockquote> <!-- 块级引用，表示引用外部内容 -->
<code>代码片段</code>   <!-- 行内代码，用于显示代码片段 -->

CSS 布局技术

现代CSS提供了多种强大的布局技术，Flexbox和Grid是其中最重要的两种。

Flexbox 布局
.container {
    display: flex;                   /* 启用Flex布局 */
    flex-direction: row;             /* 主轴方向：row | row-reverse | column | column-reverse */
    flex-wrap: wrap;                 /* 换行方式：nowrap | wrap | wrap-reverse */
    justify-content: center;         /* 主轴对齐：flex-start | flex-end | center | space-between | space-around */
    align-items: center;             /* 交叉轴对齐：stretch | flex-start | flex-end | center | baseline */
}

响应式设计

响应式设计确保网站在各种设备上都能提供最佳用户体验。

媒体查询基本语法
/* 移动设备优先 (默认样式) */
.container { 
    padding: 10px; 
}

/* 平板设备 (最小宽度768px) */
@media (min-width: 768px) {
    .container { 
        padding: 20px; 
    }
}

/* 桌面设备 (最小宽度1024px) */
@media (min-width: 1024px) {
    .container {
        padding: 30px;
        max-width: 1200px;
        margin: 0 auto; /* 居中显示 */
    }
}

CSS 颜色变量定义 
:root {
    --primary: #4361ee;    /* 主色调 - 用于按钮、链接、重要元素 */
    --primary-light: #4895ef; /* 主色调亮色 - 用于悬停状态 */
    --secondary: #3f37c9;  /* 辅助色 - 用于次要按钮、边框 */
    --light: #f8f9fa;      /* 浅背景色 - 用于卡片背景 */
    --dark: #212529;       /* 深文本色 - 主要文本颜色 */
    --success: #4caf50;    /* 成功色 - 表示成功状态 */
    --warning: #ff9800;    /* 警告色 - 表示警告状态 */
    --danger: #f44336;     /* 危险色 - 表示错误状态 */
}

按钮样式

按钮是用户交互的核心元素，需要提供清晰的视觉反馈。

基础按钮样式
.btn {
    display: inline-block;
    padding: 12px 24px;
    border: none;
    border-radius: 6px;
    background: var(--primary);
    color: white;
    font-size: 1rem;
    font-weight: 600;
    cursor: pointer;
    transition: var(--transition);
    box-shadow: var(--shadow);
}

/* 悬停效果 */
.btn:hover {
    background: var(--primary-light);
    transform: translateY(-3px);
    box-shadow: var(--shadow-hover);
}

/* 按钮状态变体 */
.btn-success {
    background: var(--success);
}
.btn-warning {
    background: var(--warning);
}
.btn-danger {
    background: var(--danger);
}

CSS 动画

CSS动画可以为用户界面添加生动的交互效果，提升用户体验。

关键帧动画
/* 定义滑入动画 */
@keyframes slideIn {
    0% {
        opacity: 0;
        transform: translateX(-100%);
    }
    80% {
        transform: translateX(10%);
    }
    100% {
        opacity: 1;
        transform: translateX(0);
    }
}

/* 应用动画到元素 */
.animated-element {
    animation: slideIn 0.8s ease-out forwards;
}
过渡效果
.box {
    width: 100px;
    height: 100px;
    background: var(--primary);
    transition: all 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275);
}

.box:hover {
    transform: rotate(15deg) scale(1.1);
    background: var(--primary-light);
}
