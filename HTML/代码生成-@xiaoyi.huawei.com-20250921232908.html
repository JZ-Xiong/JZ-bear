HTML5 语义化

创建有意义的文档结构

CSS3 动画

创建流畅的用户体验

利用CSS3的强大动画功能，无需JavaScript即可创建平滑的过渡和动画效果。

关键特性

transition, transform, animation, @keyframes, 以及硬件加速特性

.animated-box { width: 100px; height: 100px; background: #4361ee; transition: all 0.4s ease; transform: rotate(0); }
.animated-box:hover { transform: rotate(45deg) scale(1.2); background: #4cc9f0; box-shadow: 0 10px 20px rgba(0,0,0,0.2); }

响应式设计

多设备完美适配

使用媒体查询、弹性布局和网格系统创建在任何设备上都能完美显示的网站。

核心技术

Flexbox, CSS Grid, 相对单位 (rem, vw), 以及媒体查询

.responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 2rem; }
/* 平板设备 */ @media (max-width: 900px) { .responsive-grid { grid-template-columns: 1fr 1fr; } }
/* 手机设备 */ @media (max-width: 600px) { .responsive-grid { grid-template-columns: 1fr; } }
